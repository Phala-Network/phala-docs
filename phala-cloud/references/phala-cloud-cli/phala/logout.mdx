## Command: `phala logout`

### Syntax

```
phala logout
```

### Description

Remove the stored API key

### Options

| Option | Description |
| ------ | ----------- |
| `-h, --help` | Show help information for the current command |
| `-v, --version` | Show CLI version |

### Examples

* Display help:

```bash
phala logout --help
```
