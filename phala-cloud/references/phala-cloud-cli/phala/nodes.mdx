## Command: `phala nodes`

### Syntax

```
phala nodes <command> [options]
```

### Description

List and manage TEE nodes

### Options

| Option | Description |
| ------ | ----------- |
| `-h, --help` | Show help information for the current command |
| `-v, --version` | Show CLI version |

### Subcommands

| Command | Description |
| ------- | ----------- |
| `list` | List all available worker nodes |

### Examples

* Display help:

```bash
phala nodes --help
```
