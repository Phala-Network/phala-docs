If you want to deploy exclusive LLM service on GPU TEE, Confidential AI Models service is what you want.

## Choose LLM Models

Click **Confidential AI Models** tab on Phala Cloud dashboard. Choose the model you want to deployh and click button of **Deploy**.


<Frame>
    <img src="/images/confidential-ai/confidential-model/confidential-ai-models-01.png" alt="Confidential AI Models" />
</Frame>

1. Set the model service **Name** (e.g., "qwen3-8b-d7a7s")
2. Choose model in **Model template** (e.g., "qwen3")
3. **Node & Image**: Usually, leave this option for default value
4. Choose instance type in **Resource Plan**
5. Set a specific date of **Scheduled Destroy** and time when this instance will be automatically destroyed. Leave empty for no automatic destruction.
6. Finally, review the spec in **Deployment Summary** and click **Deploy** button.

<Frame>
    <img src="/images/confidential-ai/confidential-model/confidential-ai-models-02.png" alt="Deploy LLM Model" />
</Frame>

Congratulations! You exclusive confidential AI model service will be ready soon.
