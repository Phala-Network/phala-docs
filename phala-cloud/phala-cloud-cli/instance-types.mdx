## Command: `phala instance-types`

### Syntax

```
phala instance-types [options] [<family>]
```

### Description

List available instance types

### Arguments

| Argument | Description |
| -------- | ----------- |
| `<family>?` | Instance type family (cpu, gpu) |

### Global Options

| Option | Description |
| ------ | ----------- |
| `-h, --help` | Show help information for the current command |
| `-v, --version` | Show CLI version |
| `--api-token TOKEN, --api-key TOKEN` | API token used for authentication |
| `-j, --json, --no-json` | Output in JSON format |
| `--api-version <value>` | API version to use (e.g. 2025-10-28, 2026-01-21) |

### Examples

* List all instance type families

```bash
phala instance-types
```

* List CPU instance types

```bash
phala instance-types cpu
```

* List GPU instance types

```bash
phala instance-types gpu
```

* Output as JSON

```bash
phala instance-types --json
```
