## Command: `phala cvms device-allowlist`

### Syntax

```
phala cvms device-allowlist [options] [<cvm_id>]
```

### Description

Show device allowlist status for a CVM's app

### Arguments

| Argument | Description |
| -------- | ----------- |
| `<cvm_id>?` | CVM identifier (UUID, app_id, instance_id, or name) |

