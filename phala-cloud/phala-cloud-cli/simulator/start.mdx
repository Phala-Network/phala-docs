## Command: `phala simulator start`

### Syntax

```
phala simulator start [options]
```

### Description

Start the TEE simulator

### Options

| Option | Description |
| ------ | ----------- |
| `-p, --port <value>` | Simulator port (default: 8090) |
| `-v, --verbose` | Enable verbose output |
