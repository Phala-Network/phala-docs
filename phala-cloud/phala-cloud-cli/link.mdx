## Command: `phala link`

### Syntax

```
phala link [options] [<cvm-id>]
```

### Description

Link a local directory to a CVM

### Arguments

| Argument | Description |
| -------- | ----------- |
| `<cvm-id>?` | CVM ID or name (interactive if omitted) |

### Global Options

| Option | Description |
| ------ | ----------- |
| `-h, --help` | Show help information for the current command |
| `-v, --version` | Show CLI version |
| `--api-token TOKEN, --api-key TOKEN` | API token used for authentication |
| `-j, --json, --no-json` | Output in JSON format |
| `--api-version <value>` | API version to use (e.g. 2025-10-28, 2026-01-21) |

### Examples

* Link by name or ID

```bash
phala link my-cvm-name
```

* Link interactively

```bash
phala link
```
