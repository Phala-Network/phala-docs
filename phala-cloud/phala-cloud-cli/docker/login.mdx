## Command: `phala docker login`

### Syntax

```
phala docker login [options]
```

### Description

Login to Docker Hub

### Options

| Option | Description |
| ------ | ----------- |
| `-u, --username <value>` | Docker Hub username |
| `-p, --password <value>` | Docker Hub password |
| `-r, --registry <value>` | Docker registry URL |
