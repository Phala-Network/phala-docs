## Command: `phala os-images`

### Syntax

```
phala os-images [options]
```

### Description

List available OS images

### Options

| Option | Description |
| ------ | ----------- |
| `--dev` | Show only dev images |
| `--prod` | Show only prod images |
| `--page <value>` | Page number (1-indexed) |
| `--page-size <value>` | Items per page (max 100) |
| `--all` | Fetch all pages |

### Global Options

| Option | Description |
| ------ | ----------- |
| `-h, --help` | Show help information for the current command |
| `-v, --version` | Show CLI version |
| `--api-token TOKEN, --api-key TOKEN` | API token used for authentication |
| `-j, --json, --no-json` | Output in JSON format |
| `--api-version <value>` | API version to use (e.g. 2025-10-28, 2026-01-21) |

### Examples

* List all OS images

```bash
phala os-images
```

* List prod images only

```bash
phala os-images --prod
```

* List page 2 with page size 20

```bash
phala os-images --page 2 --page-size 20
```

* Fetch all pages

```bash
phala os-images --all
```

* Output as JSON

```bash
phala os-images --json
```
